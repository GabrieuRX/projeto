SE APLICAM OU NÃO A NOSSA EMPRESA ?

IP (Internet Protocol)
- Aplicabilidade: Sim, o IP se aplica diretamente
Ele garante que os dispositivos, como terminais de rastreamento, sistemas de gestão de armazéns e servidores de banco de dados, possam se comunicar de forma eficiente.
TCP (Transmission Control Protocol)
- Aplicabilidade: Sim, o TCP é altamente aplicável
É fundamental para a comunicação de dados confiáveis entre os sistemas da empresa, como ( o sistema de gerenciamento de transporte e o sistema de gestão de armazém)
IMAP (Internet Message Access Protocol)
- Aplicabilidade: Sim, o IMAP se aplica, mas indiretamente
empresas recebem confirmações de pedidos, atualizações sobre entregas, e notificações de fornecedores e transportadoras via e-mail (deixando assim os dispositivos sicronizados)
SMTP (Simple Mail Transfer Protocol)
- Aplicabilidade: Sim, o SMTP é altamente aplicável
Seria utilizado pela empresa de logística para enviar notificações automatizadas aos clientes e fornecedores, como a confirmação de pedidos
HTTP (Hypertext Transfer Protocol)
- Aplicabilidade: Sim, o HTTP se aplica diretamente
O HTTP é para a criação de portais e sistemas baseados na web, como plataformas de rastreamento de pedidos, sistemas de gestão de transporte e sites de e-commerce.
P2P (Peer-to-Peer)
- Aplicabilidade: Não, o P2P não é adequado
A descentralização e a falta de controle podem ser um risco para a integridade e a segurança dos dados, o que é crítico em uma rede que envolve monitoramento de entregas
DNS (Domain Name System)
- Aplicabilidade: Sim, o DNS se aplica diretamente
Os clientes e funcionários acessam as plataformas de forma simples apenas com o nome de domínio do site.
